**On this page**, you will need to introduce your worklog. **How** did you complete it? How many weeks did you take to complete the program? **What** did you do in those weeks?

Typically, and as a standard, a worklog is carried out over about 3 months (throughout the internship period) with weekly contents as follows:

**Week 1:** [Learn about AWS Free Tier, Billing Dashboard, IAM, MFA, EC2 and complete a mini IAM project](1.1-week1/)

**Week 2:** [Get familiar with AWS CLI, IAM roles, and basic S3 operations](1.2-week2/)

**Week 3:** [Learn EC2 in detail: Instance types, AMI, EBS, Security Groups, and SSH connection](1.3-week3/)

**Week 4:** [Practice launching EC2 instances, attaching EBS, and connecting via SSH](1.4-week4/)

**Week 5:** [Understand AWS networking: VPC, Subnets, Internet Gateway, and Elastic IPs](1.5-week5/)

**Week 6:** [Practice configuring VPC, subnet, security group rules, and EC2 networking setup](1.6-week6/)

**Week 7:** [Explore RDS, DynamoDB, and basic database configuration on AWS](1.7-week7/)

**Week 8:** [Learn S3 advanced features: versioning, lifecycle rules, bucket policies, and static website hosting](1.8-week8/)

**Week 9:** [Connect with FCJ members, review AWS basics, create AWS account, practice EC2 launch and EBS attachment](1.9-week9/)

**Week 10:** [Consolidate AWS skills: EC2, EBS, CLI commands, and Management Console usage](1.10-week10/)

**Week 11:** [Practice advanced AWS CLI commands, automate EC2 creation, and integrate basic services](1.11-week11/)

**Week 12:** [Learn CloudFormation, deploy EC2 + S3 stacks, integrate Lambda & API Gateway, test automation, and finalize internship report](1.12-week12/)
