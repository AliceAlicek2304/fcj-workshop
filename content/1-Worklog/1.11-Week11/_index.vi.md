### Mục tiêu tuần 11:

* Tìm hiểu AWS Lambda và serverless computing.
* Biết cách tạo Lambda function, trigger, quản lý version.
* Tìm hiểu API Gateway để expose Lambda function thành endpoint HTTP.
* Thực hành tích hợp Lambda với S3 và các dịch vụ AWS khác.

* Hiểu kiến trúc serverless và Lambda.
