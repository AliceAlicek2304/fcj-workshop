### Week 11 Objectives:

* Learn about AWS Lambda and serverless computing.
* Understand how to create Lambda functions, triggers, and manage versions.
* Learn about API Gateway to expose Lambda functions as HTTP endpoints.
* Practice integrating Lambda with S3 and other AWS services.
* Understand basic monitoring and logging with CloudWatch.
